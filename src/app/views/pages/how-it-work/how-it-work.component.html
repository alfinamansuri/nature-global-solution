HOW IT WORK

CTN SUBMISSION STEPS

  • Go to the website
  • Select shipper
  • Enter your username and password
  • Go to create CTN/ACD/ACI
  • Select your transaction type (Import/Export)
  • Enter the necessary information in the provided spaces
  • Upload BL and Commercial invoice
  • Submit

  • Login as Carrier
  • Enter your username and password
  • Go to create CTN/ACD/ACI
  • Select your transaction type (Import/Export)
  • Enter the necessary information in the provided spaces
  • Upload BL and Commercial invoice
  • Submit
